Puraa

A small Android app that forwards incoming SMS to a private Telegram channel or Discord channel the instant each message arrives — handy for mirroring bank, UPI, and OTP alerts to a client you already use. No second app, no server.

How to use it

  1. Install the APK on the phone whose SMS you want to forward (see the note below on install method and SMS permissions).
  2. Open Puraa and go to the relay setup screen.
  3. Tap Scan setup QR code and point at the QR above.
  4. Review the pre-filled fields, then save to start forwarding.

Installing via adb install keeps SMS permissions working; some sideload methods get SMS access blocked by Android. See the README for the details.
