Before you fly

Install the APK — the right way.

How you install matters for SMS permissions. Apps installed over adb keep SMS access; some sideload paths get it blocked by Android.

Recommended

Install with adb

adb install puraa.apk from a computer. SMS permission stays grantable, and the relay works as intended.

Heads up

Direct sideload

Opening the APK on the phone can flag it, and Android may block SMS access. See the README for the fix.
